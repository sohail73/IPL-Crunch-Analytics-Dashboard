# IPL Crunch ’26 – Data Analytics Dashboard

An interactive **Streamlit-based IPL analytics dashboard** built using real ball-by-ball IPL data.  
This project analyzes match trends, player performance, venue behavior, toss impact, and match phases using Python and modern data visualization tools.

---

## Live Features

### Home Dashboard
- Overall IPL summary (matches, teams, seasons)
- Top batters & bowlers
- Venue insights
- Toss impact overview
- Quick insights panel

---

### Toss Analysis
- Toss win vs match win probability
- Toss decision patterns (bat/field)
- Venue-wise toss impact
- Season-wise toss trends
- Team-wise toss performance

---

### Batting Analysis
- Top run scorers
- Strike rate analysis
- Boundary (4s & 6s) dominance
- Season-wise run trends
- Team batting strength
- Player run distribution

---

### Bowling Analysis
- Top wicket takers
- Economy rate analysis
- Dot ball specialists
- Season-wise wicket trends
- Team bowling performance
- Dismissal type analysis

---

### Venue Analysis
- Most used IPL venues
- Highest scoring stadiums
- Toss impact by venue
- City-wise match distribution
- Team performance at venues
- Season-wise venue trends

---

### Match Phase Analysis
- Powerplay, Middle overs, Death overs breakdown
- Phase-wise run distribution
- Wicket distribution by phase
- Over-wise scoring trends
- Death over hitters
- Powerplay wicket takers

---
